STEVDB (STellar EVolution DataBase) is a code to create a database with simulation of the
`MESA <https://docs.mesastar.org/en/release-r22.11.1>`__ stellar-evolution code

It is an open source project hosted in GitHub:
`STEVDB <https://github.com/asimazbunzel/stevdb>`__
