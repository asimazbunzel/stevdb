Contents

STEVDB (STellar EVolution DataBase manager) is a code to manage a database of different models
computed with the `MESA <https://docs.mesastar.org/>`__ stellar-evolution code

It is and open source project hosted in Github:
`STEVDB <https://github.com/asimazbunzel/stevdb>`__
